津田塾のまなび

「変革を担う女性の育成」を使命に、
人生を変える濃密な4年間を約束します

「男性と協力して対等に力を発揮できる女性の育成」創立者 津田梅子のこの想いは脈々と受け継がれ、時代を支える“新しい女性たち”の輩出へとつながりました。 津田塾大学はこれからも時代の要請に応え、新たな道を開拓していく女性を育成していきます。
